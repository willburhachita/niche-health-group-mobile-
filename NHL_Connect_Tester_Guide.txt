BEFORE YOU START
----------------
- Install the app (niche.apk) on your Android phone
- Go to Settings > Security > enable "Install from unknown sources"
- Make sure you have internet/WiFi
- You will need access to the email inbox you log in with
  (a verification code will be sent to it each time you log in)


================================================================
SECTION 1 — LOGGING IN
================================================================

The app has 3 types of staff accounts. For testing, use the
ADMIN account which has access to everything:

  Email:    wilburhachita@gmail.com
  Password: Michelle13//.

STEP-BY-STEP LOGIN:
1. Open the app. Wait for the splash/logo screen to pass.
2. Tap "Get Started" on the Welcome screen.
3. Enter the email address above and tap "Continue".
4. Check the email inbox — a 6-digit code will arrive from
   noreply@nichehealthcaregroup.com
5. Type the 6-digit code into the verification screen.
6. Enter the password above on the Password screen.
7. You should land on the Home screen.

   WHAT TO NOTE:
   - The code expires in 10 minutes. If it expires, go back
     and request a new one.
   - If the code email does not arrive, check Spam/Junk folder.

SECTION 2 — HOME SCREEN
================================================================

After logging in you will see the Home screen with:

- A greeting at the top ("Good morning / afternoon / evening")
- TODAY'S STATS cards showing:
    Total Appointments | Pending | Confirmed | Arrived | Completed
- NEXT APPOINTMENT card (the next upcoming one today)
- TODAY'S SCHEDULE list of all appointments for the day

   WHAT TO TEST:
   [ ] Stats cards display numbers (even if 0 — that is fine for
       a fresh database)
   [ ] Tapping the Next Appointment card takes you to its detail
   [ ] The bottom tab bar has 5 tabs:
       Home | Messages | Channels | Schedule | More

THINGS TO REPORT BACK
================================================================

Please take note of the following when testing and report back:

1. Any screen that crashes or shows a red/white error page
2. Any button that does nothing when tapped
3. Any data that does not save after tapping Save
4. Anything that looks visually broken (overlapping text,
   missing buttons, images not loading)
5. Any confusing wording or steps that are unclear
6. Actions that work slowly or take too long to respond

For each issue, please note:
- Which screen you were on
- What you tapped or did
- What happened vs what you expected
